# pokedex-app

Pokedex Project made in Vue.js

- API Used: https://pokeapi.co/
- Demo live: https://pokedex-app-kraiver.netlify.app/

### The Pokedex includes all Pokemons with:

- Descriptions and types
- Base statistics
- Special attacks
- Sprites
- Moves

### Functionalities:

- Pokemons List
- Pokemon Details
- Search Pokemon
- Infinity Scroll
- Responsive for phones and iPad

### Technologies used:
- Vue.js
- Sass
